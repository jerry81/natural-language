- so が -> を

- 这个单词有很多意思。
- この単語がたくさん意味はあります → WRONG
- この単語には意味がたくさあります　→ correct

- 她说什么了？
- 彼女は何言っていましたか　→ WRONG
- 彼女は何と言っていましたか　→ correct
- notice the と
- と　indicates reported speech

- たくさん仕事をして疲れました
- 很多工作做累了 - incorrect
- 做了很多工作累了 -correct
